Entrepreneurship Module Project : AIPPARELLO

Project Description

Meet AIPARELLO! Please click the image to view our website and buy a shirt *wink* :P. For this project, we created the whole website using Shopify, using metadata tags to ensure that we can add new pages using templates. The graphics that you see on the website and advertisements are made using a combination of Canva mockup tools as well as Adobe Photoshop. Furthermore, since our current mode of shirt profile redirection uses QR codes which may not be the preference of Singaporeans, I intend to use a web app that uses transfer learning to give users the chance to access their shirt profiles by accessing the web app and uploading a picture of the shirt.

Technologies Used

Shopify, TensorFlow, Adobe Photoshop
