SwingTogether

Project Description

Our submission for our school's 3.007 Design Thinking and Innovation. My primary responsibility in this project is the dynamic lighting inspired by an ArtScience museum exhibit. The lighting is powered using an ARDUINO and the presence of objects is detected using a ultrasonic sensor concealed within the top bars of the swing. The lighting changes colour when it passes through an occupied seat. When all 4 seats are occupied, there would be a momentary lightshow. More about our project here.
