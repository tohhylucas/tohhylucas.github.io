Designing Energy Systems Project

Project Description

For our Designing Energy Systems Course in Term 3 of SUTD, we decided to create a solar-powered soldering iron for our project. For this project, I had exposure to using various tools for data analysis purposes to measure the real world performance of the solar panels. Furthermore, I used Fusion360 to create the shell that would contain our all-in-one solution.
